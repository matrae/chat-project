Web Server

In the course "Software Engineering" we learn about networking, and implement a simple web browser and web server. One of the exercises is to complete the web server so that it can handle binary files (pictures), and serve multiple clients simultaneously.

If you complete this exercise, you really will have a real web server. This site is proof, because it is running on that web server! Since this is a headless server, it was started from the class WebServerNoGUI.

Project repository

The code for the web browser and web server exercises is located in the networking lecture, in the Software Engineering repository: https://gitlab.fhnw.ch/bradley.richards/WI_SoftwareEngineering.
