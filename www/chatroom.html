Chatroom

One networking project is to create a chat client that implements a specific API. The API is supported by a server, and the code for that server is provided in the repository.

For convenience, this server also provides a running instance of the chatroom server, on port 50001. As of this writing, the server has just been implemented, so there may still be bugs! If you find a problem, please let me know :-)

Project repository

The code for the chatroom server is one of the projects in the repository: https://gitlab.fhnw.ch/bradley.richards/java-projects.
